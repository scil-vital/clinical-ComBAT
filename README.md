The code on this repo is a revisited version of Jean-Philippe Fortin's [original work](https://github.com/Jfortin1/ComBatHarmonization).

The purpose of our repo is to propose a simplified version of the ComBAT method tailored to the experiments provided in the following paper

M. Edde, G.Girard, Y. David, M. Descoteaux, P-M Jodoin.  **The Does and Donts of the ComBAT Harmonization method**. Journal of Neuroimaging,X(Y):ABC-DEF, 2024.
